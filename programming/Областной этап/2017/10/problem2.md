



| Ограничение по времени      | Ограничение по памяти         |
|:----------------------------|:------------------------------|
|-|-|

Во второй строке находятся n целых числа $p_1, p_2, ..., p_n$ ($1 ≤ p_i ≤ n$) и $p_i \ne i$.










#### Примеры

Вход
```
3 
2 3 1 
5 4 8
```

Выход
```
8
```
Вход
```
5 
5 3 4 2 1 
7 6 0 3 3
```

Выход
```
7
```
Вход
```
7 
7 1 1 3 3 7 4 
3 1 4 0 2 0 2
```

Выход
```
5
```
Вход
```
12
11 1 6 1 7 8 8 7 7 9 12 11
12 29 7 19 4 8 39 31 22 34 25 40
```

Выход
```
81
```
Вход
```
10 
2 1 1 1 3 4 3 6 7 8 
2 3 1 4 0 5 1 1 3 0
```

Выход
```
12
```
